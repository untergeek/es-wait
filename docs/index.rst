Welcome to es-wait's documentation!
===================================

The es-wait package provides utilities for waiting on Elasticsearch operations,
such as index relocation, snapshot creation, and ILM phase transitions. Built on
the elasticsearch8 Python client, it offers robust polling, timeout handling, and
logging for various tasks.
